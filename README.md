# 家庭常备药与日常护理指南 (Family Medicine & Care Guide)

本仓库是一个记录日常家中常备药品、常见皮肤问题护理以及外伤应急处理知识的综合资料库。本指南各章节经过严谨编排，均附带有GB/T 7714规范的学术参考文献及顶级医学机构（如WHO、梅奥诊所等）科普链接，旨在帮助家庭建立合理的备药体系，掌握科学的用药原则，以及在遇到突发外伤或常见小毛病时，能够沉着、正确地进行初步处理。

> ⚠️ **免责声明**：本指南内容仅为日常经验总结与建议。个人体质、过敏史及特殊人群（孕妇、儿童、老人）用药存在差异。任何用药均请务必仔细阅读药品说明书或遵医嘱。危急情况请务必第一时间拨打120或前往医院急诊。

## 目录与内容导读 (Table of Contents)

### 1. 💊 [家中常备药笔记](家中常备药.md)

涵盖了家庭常规备药的核心原则与分类用药指南：
- **用药原则**：不重复用药、服药期间禁酒、特殊人群就医指南等。
- **解热镇痛药**：布洛芬与对乙酰氨基酚的使用区别及注意事项。
- **肠胃系统药物**：针对腹泻、肠胃炎、胃痛、便秘及胀气的用药推荐（如蒙脱石散、益生菌、奥美拉唑等）。
- **抗过敏及外用药物**：氯雷他定、碘伏及常见软膏的使用场景。

### 2. 🧴 [皮肤问题与护理（含备药清单）](皮肤问题与护理.md)
指导如何正确辨别和处理各类常见的皮肤异常：
- **外用备药清单**：涵盖基础保湿（凡士林）、抗过敏止痒（激素类药膏选择）、抗真菌（达克宁等）、抗细菌（百多邦等）及痤疮用药等。
- **快速鉴别表**：帮助快速区分湿疹、皮炎、真菌感染、荨麻疹、单纯疱疹等常见症状。
- **分类处理建议**：针对不同皮肤问题提供轻症处理指导及明确的就医指征。

### 3. 🚑 [外伤及应急处理（含备药清单）](外伤及应急处理.md)
提供突发意外时的紧急施救指南及急救包配置建议：
- **家庭急救箱配置**：包扎固定用品、消毒清洁剂（碘伏、生理盐水）、常用器械等。
- **烧伤/烫伤处理**：一至三度烫伤的识别，以及“降温、保护、忌涂抹偏方”的标准处理流程。
- **割伤/擦伤与止血**：浅表伤口清理消毒、大出血加压包扎法、断肢/断指的紧急保存液处理。
- **摔伤与骨折判断**：判断骨折/脱臼的常见症状，提供急救固定原则。


### 4. 🤧 [感冒及上呼吸道感染](感冒及上呼吸道感染.md)
提供关于发热、感冒及呼吸道症状的快速判断与科学护理方案：
- **感染类型鉴别**：教你快速区分普通感冒、流感、细菌性咽扁桃体炎及支原体感染。
- **用药法则**：退热、止咳、复方感冒药的“不重复”及“中西合并”等避坑原则。
- **特殊人群与护理**：婴幼儿、孕妇、老人的用药禁忌，以及科学补充体液、海盐水洗鼻等非药物干预指导。

## 如何使用本指南
- **日常查阅**：可作为按需搜索的知识库，比如遇到“拉肚子”、“长痘痘”、“做饭切到手”时，直接到对应文档运用Ctrl+F检索。
- **定期盘点**：建议每半年根据清单检查一次家中的小药箱，丢弃过期药品，及时补充耗材。
- **分享给家人**：希望家中的每位成员都能了解基本的急救和用药安全常识，关键时刻能发挥大作用。
